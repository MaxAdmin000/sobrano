## 2. Перечень персональных данных

Согласие даётся на обработку следующих персональных данных:

- Тип браузера и операционной системы (User-Agent)

- Идентификатор сессии в localStorage

## 3. Цели обработки

Оператор обрабатывает персональные данные исключительно для следующих целей:

1. **Оформление и выполнение заказа** — связь с курьером, передача данных доставки, разрешение проблемных ситуаций
2. **Учёт клиентов и истории заказов** для повторных покупок
3. **Информирование о статусе заказа** — email и/или мессенджер
4. **Маркетинговые рассылки** — *только при отдельном согласии Субъекта данных* (отдельная галочка при оформлении)
5. **Обработка обращений с формы «Контакты»**
6. **Защита Сайта** — детектирование атак, брутфорса, мошенничества
7. **Анализ посещаемости** — улучшение работы Сайта (без идентификации личности)
8. **Бухгалтерский и налоговый учёт** в соответствии с требованиями НК РФ

## 5. Передача третьим лицам

Оператор передаёт персональные данные третьим лицам **только в следующих случаях**:

1. **Курьерской службе** (своей или Яндекс Go) — для выполнения доставки. Передаются ФИО получателя, телефон, адрес.
2. **SMTP-провайдеру** — для отправки писем о статусе заказа. Передаётся email-адрес.
3. **Яндекс.Метрике** — анонимизированные данные о поведении на сайте.
4. **По требованию закона** — уполномоченным государственным органам.

Полный перечень и условия передачи указаны в **Политике конфиденциальности** (https://sobrano.store/privacy-policy.html), раздел 6.

Передача персональных данных за пределы Российской Федерации **не осуществляется**.
