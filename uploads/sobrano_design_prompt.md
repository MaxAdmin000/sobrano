# ПРОМТ ДЛЯ РАЗРАБОТКИ САЙТА «СОБРАНО — ЦВЕТОЧНАЯ КОРОБОЧКА»

---

## КОНТЕКСТ И КОНЦЕПЦИЯ

Ты создаёшь сайт для цветочного онлайн-сервиса **«Собрано»** с революционно простой идеей: клиент выбирает **размер бокса** (S / M / L / XL) с фиксированной ценой, а затем **самостоятельно наполняет его любыми цветами** — без доплат, без навязанных букетов, без сюрпризов на кассе.

Сайт — это **конструктор букета**: пошаговый, интуитивный, эстетически совершенный.

---

## ДИЗАЙН-КОНЦЕПЦИЯ И ЭСТЕТИКА

**Тон**: Organic luxury meets editorial minimalism. Как если бы Kinfolk Magazine открыл цветочный магазин.

**Визуальный язык**:
- Много воздуха, белого пространства, крупной типографики
- Цветы — главные герои: большие, несжатые, высококонтрастные фотографии
- Сдержанная палитра фона, которая не конкурирует с цветами
- Тонкие анимации, которые создают ощущение живого, дышащего продукта

**Цветовая палитра**:
- Основной фон: `#FAFAF8` (почти белый, чуть тёплый)
- Текст: `#1A1A1A` (почти чёрный, не холодный)
- Акцент: `#C4A882` (пшеничный / пыльное золото)
- Вторичный акцент: `#8B7355` (тёмный кэмел)
- Hover / активные состояния: `#2D2D2D`

**Типографика**:
- Заголовки: `Cormorant Garamond` (serif, элегантный, с характером) — 60–100px на hero
- Подзаголовки: `Cormorant Garamond Italic` — для акцентов и описаний
- Тело / интерфейс: `Neue Haas Grotesk` или `DM Sans` — чистый, нейтральный, читабельный
- Размеры боксов (S/M/L/XL): огромные буквы, 200–300px, почти абстрактные

**Анимации**:
- Плавный появление элементов при скролле (staggered fade + slight translateY)
- Hover на карточках боксов: scale 1.02 + reveal overlay с информацией
- Счётчик цветков: числа меняются с анимацией (flip или count-up)
- Переходы между шагами: горизонтальный slide или opacity + scale

## СТРУКТУРА СТРАНИЦЫ — ДЕТАЛЬНОЕ ТЗ

### ЭКРАН 1 — HERO

**Задача**: Передать концепцию за 3 секунды. Вызвать желание «хочу такой бокс».

**Структура**:
- Полноэкранный layout (100vh)
- Слева: логотип `СОБРАНО` крупно (Cormorant, 18–20px, буквенный трекинг 0.3em), под ним tagline в 2 строки: «Выбери бокс. Наполни цветами. Цена не изменится.» — максимально ёмко
- По центру или справа: большая фотография готового бокса с цветами, возможно с лёгким parallax-эффектом при скролле
- CTA-кнопка «Выбрать бокс» — якорная ссылка на экран 2. Стиль кнопки: pill-форма, акцентный цвет, 18px, hover — инверсия
- Снизу: едва заметная анимированная стрелка «скролл вниз»

**Нюансы**:
- Навигация: логотип слева, справа ненавязчивые иконки «Корзина (сумма)» и «Консультация» — плавающие поверх всех экранов на протяжении всего сайта
- Фоновый цвет `#FAFAF8`, никакого градиента на hero

---

### ЭКРАН 2 — ВЫБОР БОКСА

**Задача**: Клиент должен понять 4 варианта, их наполнение, цену и нажать на один. Это самый важный экран.

**Структура**:
- 4 карточки на весь экран, горизонтальный ряд (desktop) / вертикальный стек (mobile)
- Каждая карточка занимает `25vw` ширины, `80vh` высоты на desktop
- В дефолтном состоянии: большая фотография бокса + буква размера внизу (S / M / L / XL в Cormorant, ~120px, opacity 0.9)

**Hover / tap-состояние** (анимация 300ms ease):
- Полупрозрачный оверлей `rgba(26, 26, 26, 0.75)` поверх фото
- Крупно: буква размера белым (~150px)
- Чуть меньше: «17 стеблей» или «25 стеблей» — в зависимости от бокса
- Ещё ниже: цена `ХХХ ₽` (акцентный золотой цвет)
- Кнопка «Выбрать бокс» — белая pill-кнопка

**Боксы и наполнение**:
| Размер | Стеблей | Цена |
|--------|---------|------|
| S | 11 | ХХХ ₽ |
| M | 17 | ХХХ ₽ |
| L | 25 | ХХХ ₽ |
| XL | 35 | ХХХ ₽ |

**Логика клика**: При выборе бокса создаётся объект заказа с ID, фиксируется лимит стеблей, пользователь плавно переходит на экран 3.

### ЭКРАНЫ 3–5 — ВИТРИНА ЦВЕТОВ И ВЫБОР НАПОЛНЕНИЯ

**Задача**: Приятная, понятная витрина с живыми фото. Дать ощущение «шопинга» без перегруза.

**Структура**:

**Верхняя панель (sticky)**:
- Название шага: «Выбери цветы для бокса M»
- Прогресс-виджет: «Добавлено 7 из 17 стеблей» — визуально как progress bar или числа с анимацией
- Когда лимит заполнен — виджет меняет цвет и текст: «Бокс заполнен ✓»

**Основная сетка цветов** (desktop: 4 колонки, tablet: 3, mobile: 2):
- Каждая карточка: фото цветка (квадрат, object-fit: cover), название внизу, кнопки `−` `число` `+`
- При добавлении: лёгкая анимация карточки (scale 1.05 → 1.0) и обновление счётчика
- Карточки заблокированных (при достижении лимита) — opacity 0.4, pointer-events none

**Блок «Рекомендации флориста»** (после основной сетки):
- Заголовок: «Не знаете что выбрать? Флорист уже собрал»
- 3–4 готовые подборки карточками: «Для дома», «В подарок», «Экзотика», «Нежный»
- Каждая: фото, название, состав (розы 5 шт, хризантемы 3 шт...), итоговая цена с допами
- Кнопка «Добавить подборку» — заменяет всё что было выбрано

### ЭКРАН 6 — ДОПЫ (добавки)

**Задача**: Ненавязчиво предложить дополнения. Не «впаривать», а «завершить образ».

**Структура**:
- Заголовок: «Завершить букет?» — Cormorant Italic, крупно
- Подзаголовок мелко: «Необязательно, но красиво»
- 3 основных допа + дополнительные:

| Доп | Цена |
|-----|------|
| Классическая ваза | ХХ ₽ |
| Декоративная зелень | ХХ ₽ |
| Подкормка для цветов | Х ₽ |
| Бант | — |
| Открытка | — |

- Карточки с большими красивыми фото, чекбокс-выбор (не +/−)
- Кнопка «Перейти к оформлению» внизу

---

### ЭКРАН 7 — КОРЗИНА И ОФОРМЛЕНИЕ

**Задача**: Итоговый экран без путаницы. Клиент должен видеть что он заказывает, сколько платит и легко ввести данные.

**Левая колонка — состав заказа**:
- Выбранный бокс (размер, цена)
- Список цветов с количеством
- Список допов
- Выбор доставки: «Яндекс Go» или «Наша доставка (от 500 ₽ по районам)»
- Итоговая сумма жирно, крупно

**Правая колонка — данные пользователя**:
- ФИО*
- Телефон*
- Email
- Адрес доставки*
- Дата доставки* (datepicker)
- Канал для связи*: кнопки WhatsApp / Telegram / Звонок
- Кнопка «Оплатить через Robokassa»

**FAQ и доставка** — аккордеон под формой.

## ПЛАВАЮЩИЕ ЭЛЕМЕНТЫ (поверх всех экранов)

**Корзина** (правый нижний угол):
- Иконка корзины в круге `#1A1A1A`
- Если есть товары: рядом сумма заказа, лёгкий badge с количеством позиций
- Hover: лёгкое scale, tooltip «Ваш заказ»
- Появляется только после начала оформления (экран 2+)

**Консультация** (левый нижний угол или правый верхний):
- Маленькая кнопка «?» или иконка чата
- По клику: модалка или сайдбар с формой «Оставить номер — перезвоним»
- Ненавязчиво, не мешает основному flow

## ТЕХНИЧЕСКИЕ ТРЕБОВАНИЯ

- **Адаптивность**: Полностью адаптивный, mobile-first. Breakpoints: 375px / 768px / 1280px / 1920px
- **Оплата**: Интеграция Robokassa API (https://robokassa.com/)
- **CRM**: Подключение CRM Тильды для интернет-магазинов (или аналог в кастоме)
- **SEO**: H1 на hero, H2 на каждом блоке, alt-теги всех изображений, мета-теги, структурированные данные для товаров (schema.org/Product)
- **Юридика**: Политика конфиденциальности, согласие на обработку ПД, реквизиты ИП, пользовательское соглашение, публичная оферта
- **Производительность**: Lazy loading изображений, WebP формат, score Lighthouse 90+
- **Корзина**: Состояние в localStorage или sessionStorage, не сбрасывается при перезагрузке
